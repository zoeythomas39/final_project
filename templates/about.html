ABOUT THE SITE

PURPOSE

As a University of Florida media production student, I built this site for my course Advanced Web Applications for Communicators. When tasked with creating a web app incorporating scraping, databases and Python's Flask application framework, I decided to create a site based on one of my favorite artists, Elliott Smith.

Similar web apps exist for other singers — like this one for Taylor Swift from designer Margaux. But I couldn't find an existing lyric searcher for Smith online, perhaps due to the relative smallness of his fanbase compared to Swift's.

I also wanted my app to allow a user to search by album, year, keyword or title, rather than just lyrics.

METHODOLOGY

I began by scraping song information from Genius' Elliott Smith page using Python's Beautiful Soup library. Going album by album, I took the title, album, release date, Genius link and lyrics for each song. Then, I marked songs containing certain keywords with dummy indicators and wrote the results into a CSV. To make the data cleaner and avoid duplicates, I removed live versions of songs, as well as covers, demos, alternate and remastered recordings.

I lightly cleaned the CSV in Microsoft Excel — parsing the year from the "date" column, adding an index column and adjusting a few of the album titles. Because several compilation albums have been released following Smith's death, I wanted to ensure each track was listed under the album on which it originally appeared.

Next, I connected the CSV to a MySQL database in Python and created Flask templates to hold my pages. Using Bootstrap and the open-source Bootswatch, I designed templates for the home, song list and song information pages, as well as error routes.
